## 模型框架
> 此项目主要包含`utils`、`models`、`config`、`loss`四大单元模块
> + **models文件夹**：编写基本的模型结构，目前实现`convlstm`结构
> + **utils文件夹**：用于存放辅助代码
> + **loss文件夹**: 存放实现的损失函数， 目前补充**加权MAE/MSE**
> + **config文件夹**: 存放配置文件
